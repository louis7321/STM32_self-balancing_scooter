這是平衡車初始版本 雲科嵌入式實作的期末專題
 1. 硬體架構
   * MCU: STM32F103C8T6。
   * 馬達控制: 使用 TB6612FNG，透過 TIM1 (PA8, PA11) 輸出 PWM，並由 PB12-PB15 控制方向。
   * 感測器: MPU6050，使用軟體 I2C (PB8, PB9)，目前設定為繞 Y 軸 控制平衡。

  2. 軟體進度與演算法
   * 已實作互補濾波計算姿態。
   * 已實作 PD 控制器 (100Hz)，預設參數為 Kp=45.0, Kd=1.2，目標平衡角度為 -2.5 度。
   * 具備安全保護機制：傾斜超過 45 度或 PWM 超過 80% 會自動停止。
