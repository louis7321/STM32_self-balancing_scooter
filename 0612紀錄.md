# 平衡車專案開發紀錄 (2026-06-15 更新)

## 1. 硬體規格
*   **MCU**: STM32F103C8T6
*   **馬達驅動**: TB6612FNG
*   **馬達**: 25型減速馬達 (6V, 133RPM, 1:45)
*   **感測器**: 
    *   MPU6050 (陀螺儀/加速度計)
    *   TCRT5000 循跡感測器 x2

## 2. 接線配置 (Pinout)

### TB6612FNG 馬達驅動
| 功能 | STM32 腳位 | 備註 |
| :--- | :--- | :--- |
| **PWMA** | PA8 | 左輪速度 (TIM1_CH1) |
| **PWMB** | PA11 | 右輪速度 (TIM1_CH4) |
| **AIN1** | PB12 | 左輪方向控制 |
| **AIN2** | PB13 | 左輪方向控制 |
| **BIN1** | PB14 | 右輪方向控制 |
| **BIN2** | PB15 | 右輪方向控制 |
| **STBY** | PB5 | 高電位啟動驅動器 |

### MPU6050 感測器 (軟體 I2C)
| 功能 | STM32 腳位 | 備註 |
| :--- | :--- | :--- |
| **SCL** | PB8 | 軟體模擬 I2C 時鐘 |
| **SDA** | PB9 | 軟體模擬 I2C 資料 |

### TCRT5000 循跡感測器
| 功能 | STM32 腳位 | 備註 |
| :--- | :--- | :--- |
| **左感測器 (L-DO)** | PB10 | 黑線觸發為 0 (燈滅) |
| **右感測器 (R-DO)** | PB11 | 黑線觸發為 0 (燈滅) |

## 3. 軟體實作進度
1.  **馬達控制 (`Motor_Control`)**:
    *   支援正負轉與速度控制 (-1000 ~ 1000)。
    *   **安全保護**: PWM 限制在 80% (800) 以內。
    *   **傾倒保護**: 角度超過 +/-45 度自動停機。
2.  **姿態計算**:
    *   MPU6050 互補濾波，控制軸向為 **Y 軸 (Pitch)**。
3.  **平衡演算法**:
    *   實作 PD 控制迴圈 (100Hz)。
    *   **最新參數**: `Kp = 45.0`, `Kd = 1.2`, `Target_Angle = -2.3` (微調後的速度平衡點)。
4.  **循跡整合 (2026-06-15)**:
    *   實作轉向補償 `Steer_PWM`，預設力道 `Track_Speed = 127`。
    *   邏輯：左感測偵測到黑線則向左修正，右感測偵測到黑線則向右修正。

## 4. 目前問題與待辦事項
*   **調試**: 持續觀察賽道表現，必要時微調 `Track_Speed` 與 `Target_Angle`。
*   **編碼器**: 尚未整合馬達編碼器讀取 (用於速度環控制)。
