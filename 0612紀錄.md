# 平衡車專案開發紀錄 (2026-06-12)

## 1. 硬體規格
*   **MCU**: STM32F103C8T6
*   **馬達驅動**: TB6612FNG
*   **馬達**: 25型減速馬達 (6V, 133RPM, 1:45)
*   **感測器**: MPU6050 (陀螺儀/加速度計)

## 2. 接線配置 (Pinout)

### TB6612FNG 馬達驅動
| 功能 | STM32 腳位 | 備註 |
| :--- | :--- | :--- |
| **PWMA** | PA8 | 左輪速度 (TIM1_CH1) |
| **PWMB** | PA11 | 右輪速度 (TIM1_CH4) |
| **AIN1** | PB12 | 左輪方向控制 |
| **AIN2** | PB13 | 左輪方向控制 |
| **BIN1** | PB14 | 右輪方向控制 |
| **BIN2** | PB15 | 右輪方向控制 |
| **STBY** | PB5 | 高電位啟動驅動器 |

### MPU6050 感測器 (軟體 I2C)
| 功能 | STM32 腳位 | 備註 |
| :--- | :--- | :--- |
| **SCL** | PB8 | 軟體模擬 I2C 時鐘 |
| **SDA** | PB9 | 軟體模擬 I2C 資料 |
| **VCC** | 3.3V | |
| **GND** | GND | 需與系統共地 |

## 3. 軟體實作進度
1.  **馬達控制 (`Motor_Control`)**:
    *   支援正負轉與速度控制 (-1000 ~ 1000)。
    *   **安全保護**: PWM 限制在 80% (800) 以內。
    *   **傾倒保護**: 角度超過 +/-45 度自動停機並關閉驅動器。
2.  **MPU6050 驅動**:
    *   實作軟體 I2C 通訊協定。
    *   **姿態計算**: 採用互補濾波融合加速度計與陀螺儀資料。
    *   **控制軸向**: 已修改為繞 **Y 軸** 旋轉控制前後傾斜 (Pitch)。
3.  **平衡演算法**:
    *   實作 PD 控制迴圈 (100Hz)。
    *   預設參數: `Kp = 45.0`, `Kd = 1.2`, `Target_Angle = -2.5`。

## 4. 目前問題與待辦事項
*   **編譯錯誤**: 發生 `L6218E: Undefined symbol` 錯誤，需在 Keil 中將 `mpu6050.c` 手動加入專案樹。
*   **調試**: 需燒錄後觀察角度輸出方向，並進行 PID 參數實車調優。
*   **編碼器**: 尚未整合馬達編碼器讀取 (用於速度環控制)。
